CodeHub is a **full-stack repository hosting and management platform** inspired by GitHub.  
It enables developers to **create, manage, and share repositories** with authentication, Git command execution, and social features like following users.  

🔗 **Live Demo**: [CodeHub on AWS Amplify](https://main.d1jqjadpadvcks.amplifyapp.com/auth)  
📂 **Source Code**: [GitHub Repository](https://github.com/Akash-75-76/CodeHub)  

---

## ✨ Features  

- 🔐 **Authentication** – Secure user login & signup  
- 📁 **Repository Management** – Create, update, and manage repositories  
- ⚡ **Git Commands** – Execute Git-like operations inside the platform  
- 👥 **Social Features** – Follow/unfollow users  
- ☁️ **Cloud Integration** – AWS S3 for file storage  
- 🖼️ **Profile Pictures** – Upload with Multer  
- 🔒 **SSL-secured Deployment** – Nginx reverse proxy + DuckDNS SSL  

---

## 🛠️ Tech Stack  

**Frontend:** React.js, AWS Amplify  
**Backend:** Node.js, Express.js, MongoDB, Yargs  
**Cloud & DevOps:** AWS EC2, AWS S3, Amplify, Elastic IP, Nginx  
**Others:** Multer, REST APIs, JWT Authentication
